## 2. Abstraction Model

La estructura semántica DEBE ser estricta:

*   **ARC (Architecture)**: Lógica conceptual y de gobierno.
*   **BLP (Blueprint Templates)**: Plantillas reutilizables, NO contienen datos reales de un proyecto.
*   **MTX (Data Matrices)**: Matrices de datos pobladas. Esta es la única capa con datos reales del usuario.
*   **AUT (Automation Executables)**: Scripts de ejecución y automatización.
