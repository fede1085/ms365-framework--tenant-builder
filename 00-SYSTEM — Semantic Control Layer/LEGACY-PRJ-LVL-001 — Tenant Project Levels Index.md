# PRJ-LVL-001 — Tenant Project Levels Index

## Overview

High-level reference of tenant maturity stages.

---

| Level | Name                         | Description                                  |
|------|------------------------------|----------------------------------------------|
| 0    | Raw Tenant                   | Empty environment                            |
| 1    | Identity Baseline            | Users and access exist                       |
| 2    | Operational Structure        | Company operates                             |
| 3    | Governance & Control         | Structured and controlled                    |

## One-Line Meaning

- L0 → Nothing exists  
- L1 → People exist  
- L2 → Work happens  
- L3 → Work is controlled  
- L4 → Work is automated  
- L5 → Work is secure
